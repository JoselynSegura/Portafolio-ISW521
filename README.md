# Portafolio ISW-521

Repositorio personal del curso **ISW-521: Desarrollo de Aplicaciones Web**.

## Contenido

### Unidad I - Componentes de las Aplicaciones Web

#### Clase 1 - 19/05/2026

* Creación del repositorio del portafolio.
* Presentación del curso y metodología de trabajo.
* Introducción a Internet y la Web.
* URL y dominios de Internet.
* Diferencias entre Internet, Intranet y Extranet.

#### Clase 2 - 21/05/2026

* Protocolos TCP/IP.
* HTTP y HTTPS.
* TLS y seguridad en la comunicación web.
* Ciclo completo de una solicitud HTTP.
* Análisis de tráfico con DevTools (Network).

#### Clase 3 - 26/05/2026

* Servidores web.
* Host y puertos.
* Concepto de Serverless.
* Aplicaciones web y publicación de sitios.
* Introducción a CORS.
* Configuración del entorno de desarrollo.

#### Clase 4 - 28/05/2026

* Control de versiones con Git y GitHub.
* Comandos básicos de Git.
* Ramas, merge y Pull Requests.
* Introducción a npm.
* Introducción a Vite.
* Creación de la estructura inicial del portafolio.

## Tecnologías utilizadas

* Git
* GitHub
* Visual Studio Code
* Node.js
* npm
* Vite
* DevTools del navegador
* PowerShell

## Objetivo de la Unidad

Registrar el progreso y las actividades desarrolladas durante la Unidad I, fortaleciendo los conocimientos fundamentales sobre tecnologías web, protocolos de comunicación, herramientas de desarrollo y control de versiones.
